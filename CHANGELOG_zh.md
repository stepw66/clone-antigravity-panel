# 更新日志

所有对 "Antigravity Panel" 扩展的显著更改都将记录在此文件中。

## [1.0.0] - 2025-12-09

### 变更
- **发布**: 首个稳定版本，包含所有英文注释和 MIT 许可证。
- **代码质量**: 所有代码注释已更新为英文，以便更好地进行国际协作。

## [0.2.1] - 2025-12-09

### 新增
- **配额使用历史**: 在侧边栏新增图表，展示过去 24 小时的配额使用趋势。
- **配额策略系统**: 引入基于 `quota_strategy.json` 的配置驱动模型分组，支持灵活的 AI 模型分类（如 Gemini 对比 Other）。
- **多语言支持**: 为配置界面添加了 11 种语言的完整本地化支持（中、英、日、法、德、西、葡、意、韩、俄）。
- **单元测试套件**: 添加了覆盖核心逻辑的全面单元测试（113 个测试），采用基于 Mock 的架构以实现快速执行。
- **全局规则快捷方式**: 添加了 [📏 Rules] 快速访问按钮，可直接打开 `~/.gemini/GEMINI.md`。
- **配置**: 新增 `gagp.historyDisplayMinutes` 用于控制历史图表的时间显示范围。
- **架构**: 引入 `QuotaHistoryManager` 用于持久化历史追踪。
- **网络**: 实现了健壮的 HTTP 客户端，支持自动 HTTPS 到 HTTP 的降级处理。

### 变更
- **UI**: 改进了侧边栏布局以适配新的历史图表，并支持动态模型分组显示。
- **性能**: 通过使用缓存数据进行即时渲染，优化了启动性能。
- **状态栏**: 警告颜色现在遵循 `gagp.quotaWarningThreshold` 和 `gagp.quotaCriticalThreshold` 配置。
- **身份**: 更新发布者和仓库 URL 为 `n2ns` / `n2n-studio`。

## [0.2.0] - 2025-12-08

### 新增
- **UI**: 添加基于 Webview 的侧边栏面板，替代简单的 QuickPick 菜单。
- **可视化**: 添加 Gemini 和 Other 模型配额的双饼图展示。
- **Brain 管理**: 添加 Brain 任务树视图，支持文件浏览和删除功能。
- **缓存**: 添加代码上下文缓存 (`code_tracker`) 管理。
- **快捷方式**: 添加 MCP 配置和浏览器白名单的快速访问按钮。
- **图标**: 迁移至 VS Code Codicons 以获得原生外观体验。
- **异步**: 将文件操作重构为非阻塞式，并添加加载动画。

### 修复
- 通过实现统一的 `Scheduler` 解决了重复轮询间隔的问题。
- 通过实现状态持久化修复了“需重启以更新”的问题。

## [0.1.0] - 2025-12-08

- 初始发布 (MVP)。
- 状态栏基本配额监控。
- 缓存大小计算和清理命令。
- Antigravity IDE 进程检测。
